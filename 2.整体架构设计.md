# 整体架构设计

## 系统架构图

```C++
Server
  网络模块
  	TCP模块：TCPNet
  	KCP模块：KCPNet
  数据库模块：
  	数据库核心：CMySql
  	连接池：mysqlconnpool
  业务逻辑模块：core
  协议模块：
  	协议包：packdef
  	协议发送模版：packetbuilder
  常用全局变量模块：tou

client
  网络模块
  	TCP模块：TCPNet
  	KCP模块：KCPNet
  网络-逻辑交换模块：core
  游戏逻辑模块：gameclient
  物品模块：
		物品：item
  	物品工厂类：itemfactory
  登录模块：login
  聊天模块：chat
  npc模块：ncp
  玩家模块：player
  背包模块：playerbag
  地图模块：tilemap
  UI模块：widget
  物品JSON数据：item.JSON
  地图数据：data
```

## 数据流说明

### 服务器启动流程：

```mermaid
flowchart TD
    A[初始化数据库] --> B[初始化 Core]
    B --> C[启动 TCP 模块]
    C --> D[启动 KCP 模块]
    D --> E[初始化 ChatHistory]
    E --> F{是否全部成功?}
    F -->|是| G[服务器启动完成]
    F -->|否| H[服务器启动失败]
```

### 客户端启动流程：

```mermaid
flowchart TD
    Start([客户端启动])

    Start --> A[初始化游戏逻辑模块]
    A --> B[初始化 Core 与 Login 模块]
    B --> C[初始化 TCP 模块]
    C --> D{TCP 初始化成功?}

    D -->|否| F[客户端启动失败]
    D -->|是| E[发起 KCP 连接请求]

    E --> G[启动心跳定时器]
    G --> H[显示登录界面]
    H --> I[连接信号与槽]
    I --> Run([进入运行状态])
```

### 客户端-服务器登录注册流程：

```mermaid
flowchart TD
    Start([客户端启动])
    Start --> A[显示登录界面]

    A --> B{选择操作}

    %% ================= 登录流程 =================
    B -->|登录| C[输入用户名与密码]
    C --> D[发送登录请求至服务器]
    D --> E[服务器校验账号与密码]
    E --> F{登录是否成功?}

    F -->|否| I[提示登录失败]

    F -->|是| G[接收登录成功响应]
    G --> H[进入游戏主界面]

    %% ===== 登录成功后的初始化流程 =====
    H --> Q[请求玩家数据（背包/角色信息）]
    Q --> R[初始化游戏UI]
    R --> S[初始化聊天模块]
    S --> P[创建玩家实体]
    P --> Run([进入游戏运行状态])

    %% ================= 注册流程 =================
    B -->|注册| J[输入用户名、密码、电话]
    J --> K[发送注册请求至服务器]
    K --> L[服务器校验账号与电话唯一性]
    L --> M{注册是否成功?}

    M -->|是| N[提示注册成功]
    M -->|否| O[提示注册失败]
```

### 背包初始化流程：

```mermaid
flowchart TD
    Start([背包初始化])

    Start --> A[客户端发送背包初始化请求（携带玩家ID）]
    A --> B[服务器查询 player_bag 表]
    B --> C[服务器返回物品ID与数量列表]

    C --> D[客户端解析响应数据]
    D --> E[根据物品ID读取本地JSON配置]
    E --> F[构建背包数据结构]
    F --> G[刷新背包UI]

    G --> End([背包初始化完成])
```
